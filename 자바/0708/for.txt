반복문

for : for문은 가장많이 사용되는 반복문입니다.
      while, do~while문은 조건식만 있는데 반해
      for문은 초기식, 조건식, 증감식으로 구성되어 있습니다.
      (while 문은 도는 수가 불명확, for문은 명확하다)

형식

      1         2          4
for(초기식  ;  조건식   ;  증감식 )
{
	문장 1;
	문장 2;    3
}

1 => 2 => 3 => 4 => 2 => 3 => 4 => 3 => 4 => 234 반복

while문
 주어진 조건이 참이면 동일한 문장을 반복 수행합니다.
 만일 조건이 거짓이라면 지정된 블럭을 빠져나갑니다.

 형식
{ 
 문장1;
 문장2;
 ....
 }

 while문은 먼저 조건 비교하고 참인 경우에만
 while 블럭을 처리합니다.

 do ~ while문

 : while 문은 먼저 조건을 비교한 다음에 참일 경우에만
   문장을 수행하는데 반하여 do~while문은 문장을 실행한
   다음에 조건을 비교합니다.

   형식

   do {
	문장1;
	문장2;
	...;
	} while(조건) ; // ;잊지말자
