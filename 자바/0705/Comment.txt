표현식과 문장

: Expressions(식, 수식)은 연산자와 피연산자로 구성됩니다.
  연산자로 피연산자를 처리하여 새로운 값을 만들어 내는 것을
  표현식이라고 하고 상수와 변수를 조작하는 형태입니다.
  표션식의 구성 요소는 상스, 변수, 연산자 입니다.
  자바의 모든 표현식은 값을 갖습니다.
  즉 연산을 통해서 표현식이 값을 갖게 되고 이때 사용되는 ㄱ서이
  연산자 입니다.

     a   =   b   +   55;
     |       |        | 
     |          표현식   
     | 표현식       | 


문장 (Statement) : 문장이란 프로그램 실행을 위한 
                  최소 구성단위입니다.
		  자바의 모든 문장은 (; => 세미콜론)
		  으로 끝납니다.


		 
	float ff;  => 문장
	ff = 12 / 4 ; => 문장

자바의 문장은 1개 이상의 표현식으루 구성될 수 있습니다.

주석

: Comment는 프로그램 실행과는 상관없이 코드에 설명을
  붙이기 위해 사용하는 문장

  // : 한줄 주석
  /*~*/ : 여러줄 주석

특수문자 출력     의미

\n             enter의 기능을 가지며 줄을 바꾼다(New line)
\t             tab 키로 일정한 간격을 띄운다 (horizontal tab)
\b             백 스페이스 기능을 한다(backspace)
\r             동일한 줄의 맨 앞으로 커서만 옮긴다(carriage return)
\f             출력 용지를 한 페이지 넘긴다 (form feed)
\a             경고음이 난다(alert)
\\             \ 문자를 출력한다(back slash)
\'             ' 문자를 출력한다 (single quote)
\"             " 문자를 출력한다 (double quote)
\0             null 문자
